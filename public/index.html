Was ist Krypex?

Krypex ist ein einzigartiger Minecraft Server für die Java Edition mit dem innovativen FreeBuild-Spielmodus. Hier kannst du dein eigenes Land gründen, dein Territorium durch Claims erweitern und epische Kriege gegen andere Nationen führen!

Mit einem durchdachten Questsystem, individuellen Skills und einem vielfältigen Jobsystem bietet Krypex langfristigen Spielspaß und zahlreiche Möglichkeiten zur Weiterentwicklung deines Reichs.

So trittst du bei

Folge diesen einfachen Schritten, um dem Krypex Minecraft Server beizutreten

1

Minecraft öffnen

Starte Minecraft Java Edition und gehe zu "Mehrspieler". Klicke dann auf "Server hinzufügen".

2

Server hinzufügen

Gib unsere Server-Adresse ein:

Krypex.net

Klicke auf "Fertig" um den Server zu speichern.

3

Abenteuer beginnen!

Klicke auf den Krypex Server und dann auf "Server beitreten". Willkommen in deinem neuen Königreich!
